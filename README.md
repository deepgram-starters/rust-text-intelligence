# Rust Text Intelligence

Get started using Deepgram's Text Intelligence with this Rust demo app

## Quick Start

Click the button below to fork the repo:

[![Fork on GitHub](https://img.shields.io/badge/Fork_on_GitHub-blue?logo=github)](https://github.com/deepgram-starters/rust-text-intelligence/fork)

## Local Development

### Makefile (Recommended)

```bash
make init
cp sample.env .env  # Add your DEEPGRAM_API_KEY
make start
```

Open [http://localhost:8080](http://localhost:8080) in your browser.
